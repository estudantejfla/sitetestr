Sobre o Projeto

O Criar Time Pokémon é um site educativo para montar times de Pokémon, compartilhar com amigos e explorar estratégias. Projeto desenvolvido por João Felipe Lopes e Felipe Luan como parte da disciplina.

Objetivo

Praticar HTML5, CSS3 e JavaScript com foco em responsividade, interatividade (localStorage) e experiência do usuário.

Tecnologias

  • HTML5 + CSS3 (layout responsivo)
  • JavaScript (manipulação de DOM e armazenamento local)
  • PokeAPI (opcional, para imagens e dados no futuro)
  • Hospedagem: GitHub Pages, Netlify ou Vercel
